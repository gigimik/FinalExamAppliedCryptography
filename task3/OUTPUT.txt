-------------------------------------------------------------------
How TLS Provides Security
-------------------------------------------------------------------

Confidentiality:
TLS encrypts application data using symmetric encryption after
the handshake. In this session, the cipher
TLS_AES_256_GCM_SHA384 provides strong confidentiality.

Integrity:
Integrity is provided by authenticated encryption (AES-GCM),
which detects any modification of encrypted data.

Authentication:
Authentication is achieved using X.509 certificates and the
public key infrastructure (PKI). The server proves its identity
through a certificate chain signed by a trusted CA.
